Basic HTML Elements

This is an intro to some basic html Element type and their descriptions.

Element Type Element Description
Body <body></body> <body> — is a key HTML element. Only content inside the body tag is shown.
Headings <h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6> 		<h1> — used for main headings. All other smaller headings are used for subheadings
Displaying Text <p></p> Paragraphs <p> contain a block of plain text.
<span></span> <span> contains short pieces of text or other HTML. They are used to separate small pieces of content that are on the same line as other content.
Styling Text <em></em> The <em> tag will generally render as italic emphasis.
<strong></strong> The <strong> will generally render as bold emphasis.
Unordered List <ul>
<li></li>
</ul>		 an unordered list tag <ul> is used to create a list of items in no particular order. An unordered list outlines individual list items with a bullet point.
Ordered List <ol>
<li></li>
</ol>		 Ordered lists <ol> are like unordered lists, except that each list item is numbered.
Line breaks <br> The line break element is used for breaking a line of for a block of text.
Images <img src=""/> The <img> tag allows you to add an image to a web page. Most elements require both opening and closing tags, but the <img>tag is a self-closing tag. It requires a src attribute with a link to the image source.
Videos <video></video> The <video> tag allows you to add a video to a web page. It requires a src attribute with a link to the video source.

Semantic HTML

Element Type Element Description
Header <header></header> A <header> is a container usually for either navigational links or introductory content containing <h1< to >h6< headings. It can also hold the <nav> element.
Main <main></main> The element <main> is used to encapsulate the dominant content within a webpage.
Footer <footer></footer> The content at the bottom of the subject information is known as the footer, indicated by the <footer> element.
Navigation Links <nav></nav> A <nav> is used to define a block of navigation links such as menus and tables of contents. It is important to note that <nav> can be used inside of the <header> element but can also be used on its own.
Section <section></section> <section> defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
Article <article></article> <article> holds content that makes sense on its own such as articles, blogs, comments, etc.
Aside <aside></aside> The <aside> element is used to mark additional information that can enhance another element but is not required in order to understand the main content.
Figure <figure></figure> <figure> is an element used to encapsulate media such as an image, illustration, diagram, code snippet, etc, which is referenced in the main flow of the document.
figure Caption <figcaption></figcaption> <figcaption> is an element used to describe the media in the <figure> tag. Usually, <figcaption> will go inside <figure>.
Date and Time <time></date> The <time> element represents either a time of day or a calendar date.
Address <address></address> The <address> element is like <time> in that it does not deal with the overall structure of a document, but rather embellishes the parent <article> or <body>element with some metadata.
